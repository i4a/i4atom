# i4atom package

Our workflow in atom

## Installation

1. Clone [this repository](https://github.com/i4a/i4atom)

```
git clone git@github.com:i4a/i4atom.git
```


2. Go to your local atom packages folder

```
cd ~/.atom/packages
```

3. Link the package

```
ln -s <path/to/cloned/i4atom>
```

4. Restart atom

5. Configure `i4atom`
